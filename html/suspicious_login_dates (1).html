Suspicious Login Dates

Project Overview

This project involved analyzing login data to investigate suspicious activity on specific dates, particularly May 8 and May 9, 2022. By filtering login attempts for these dates, I identified patterns that warranted further investigation.

SQL Query

SELECT *
FROM log_in_attempts
WHERE login_date = '2022-05-08'
   OR login_date = '2022-05-09';

Explanation

The query retrieves all records from the log_in_attempts table where:

  • The login_date is either '2022-05-08' or '2022-05-09'.

This approach ensures that any login attempts on these dates are captured for analysis.

Results

The query returned login attempts for the specified dates, which were analyzed further to identify any suspicious patterns or unauthorized access.

Impact

This analysis helped the organization to:

  • Understand login trends during the suspicious dates.
  • Take preemptive actions to mitigate potential security threats.
