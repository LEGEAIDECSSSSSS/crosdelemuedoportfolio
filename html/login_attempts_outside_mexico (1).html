Login Attempts Outside Mexico

Project Overview

This project analyzed login attempts outside of Mexico to identify potentially unauthorized access. By filtering out logins from locations other than Mexico, the organization could monitor for suspicious activity.

SQL Query

SELECT *
FROM log_in_attempts
WHERE country NOT LIKE 'MEX%';

Explanation

The query retrieves all records from the log_in_attempts table where:

  • The country does not start with 'MEX', which includes variations like "MEXICO".

This filtering ensures only logins outside of Mexico are analyzed.

Results

The query identified several login attempts from locations outside of Mexico. These were flagged for further investigation.

Impact

The organization used this analysis to:

  • Detect unauthorized access from foreign locations.
  • Strengthen geo-based access controls to improve security.
